# Tauri + rspc + Prisma

A minimal Tauri + rspc + Prisma example app.

## Setup

```bash
# Install dependencies
pnpm install

# Generate Prisma client
pnpm prisma:generate

# Run the app
pnpm tauri dev
```

## Changes to make

- Update `tauri-rspc-prisma` and `tauri_rspc_prisma` to your app name.
